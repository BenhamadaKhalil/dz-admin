# dz-admin

Modern Algeria administrative divisions dataset and API for 2026/2027.

## Features

- 58 wilayas
- Multilingual support
- ISO codes
- Coordinates
- GeoJSON-ready
- REST API
- TypeScript SDK

{
  "code": "16",
  "name": {

  }
}
